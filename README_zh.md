## 介绍

Zeroflx是一款现代、跨平台的视频播放器，播放内核使用MPV和VLC，兼容几乎所有视频格式。支持跨设备配置同步，界面简洁美观。我们的目标是提供流畅和一致的视频播放体验。

如果遇到问题或者有功能需求，欢迎提交issue，您的支持和反馈是我们最大的提升动力。
